# FlavorFusion - Recipe Sharing Platform

A full-stack recipe sharing platform built with the MERN stack (MongoDB, Express, React, Node.js) following MVC architecture.

## Features

- 🔐 **Authentication**: JWT-based secure login and registration
- 📝 **Recipe Management**: Create, read, update, delete recipes
- 🖼️ **Media Upload**: Upload recipe images and videos using Multer
- ❤️ **Interactions**: Like, rate (1-5 stars), and comment on recipes
- ⭐ **Favorites**: Save favorite recipes to your personal collection
- 📅 **Meal Planner**: Plan meals for the week with no duplicate days
- 🛒 **Shopping List**: Auto-generated from meal plans
- 👤 **User Profiles**: Customize profile with bio and profile image
- 🔍 **Search & Filters**: Search recipes with cuisine, meal type, and dietary filters
- 📱 **Responsive Design**: Mobile-first with Tailwind CSS

## Tech Stack

### Backend
- Node.js + Express.js
- MongoDB + Mongoose
- JWT Authentication
- bcryptjs for password hashing
- Multer for file uploads

### Frontend
- React.js (Vite)
- React Router DOM
- Axios for API calls
- Tailwind CSS
- Lucide React for icons
